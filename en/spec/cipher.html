Cipher


Recommended

The following AEAD ciphers are recommended.

NameKey SizeIV LengthTag Length
xchacha20-ietf-poly1305322416
chacha20-ietf-poly1305321216
aes-256-gcm321216
aes-192-gcm241216
aes-128-gcm161216

Deprecated

The following stream ciphers still work and provide reasonable safety margin. However, users should use the above recommended AEAD ciphers if possible.

NameKey SizeIV Length
aes-128-ctr1616
aes-192-ctr2416
aes-256-ctr3216
aes-128-cfb1616
aes-192-cfb2416
aes-256-cfb3216
camellia-128-cfb1616
camellia-192-cfb2416
camellia-256-cfb3216
chacha20-ietf3212

Dangerous

The following stream ciphers are vulnerable to various attacks. Please DO NOT use them. Implementors are advised to remove them as soon as possible.

NameKey SizeIV Length
bf-cfb168
chacha20328
salsa20328
rc4-md51616
